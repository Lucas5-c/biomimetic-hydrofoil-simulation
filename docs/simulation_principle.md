# 统一仿真平台原理说明

## 1. 平台定位

本平台是仿生分区主动水翼的轻量化可视化仿真工具，服务于专利展示、汇报演示和概念验证。它不是高精度 CFD，不求解完整 Navier-Stokes 方程，不替代 OpenFOAM、Fluent、COMSOL 或 STAR-CCM+。

## 2. 三栏交互逻辑

新版界面采用三栏独立滚动：

- 左侧显示当前模块专属参数；
- 中间显示当前图像或 Canvas 动画，并在图下放置当前模块导出按钮；
- 右侧显示精密统计、控制建议、JSON 预览和最近导出记录。

这种布局让用户在调参、观察图像、读取结果时互不干扰，更接近专业仿真软件工作台。

## 3. 水翼几何

水翼截面使用 NACA 0012 近似轮廓，弦长归一化为 1。攻角变化通过坐标旋转表达。数值图中水翼内部区域被 mask，不显示压力或速度场。

## 4. 简化速度场

速度场由以下部分叠加：

- 背景来流速度；
- 攻角导致的上下表面速度差；
- 水翼近表面绕流贴附项；
- A/B/C 分区扰动项；
- 后缘尾迹和简化涡旋项。

该速度场是经验可视化模型，不是湍流模型或边界层求解器。

## 5. 压力估算

压力使用简化伯努利关系估算：

```text
p_relative = 0.5 * rho * (U_ref^2 - V^2)
```

输出单位为 kPa。它只能表示趋势，不能作为真实水动力载荷或空化裕度的定量结论。

## 6. 动态粒子流原理

动态水流粒子模块使用 HTML5 Canvas 和 `requestAnimationFrame`。粒子从左侧连续进入，按前端 JavaScript 中的简化速度场移动。

参数响应方式：

- 流速增大：粒子速度加快，拖尾变长，后缘尾迹拉长；
- 攻角增大：水翼姿态旋转，上表面粒子更贴附，尾迹向下偏转；
- A 区角度增大：前缘粒子出现更明显的小幅波纹扰动；
- B 区角度增大：中弦区域粒子明显偏转，并出现局部扰动云；
- C 区整流增强：尾迹变窄、涡旋被削弱；C 区关闭或角度较小时尾迹更发散。

粒子颜色按速度映射：低速深蓝，中速青色，高速白色或淡黄。拖尾采用透明渐隐，增强流动方向感。

## 7. A/B/C 分区扰动含义

- A 区：前缘调节区，表达边界层预扰动和快速小幅响应；
- B 区：中弦主控区，表达主控叶片对低压区、空化风险和失速趋势的主要影响；
- C 区：后缘整流区，表达尾迹收束、减阻和涡旋削弱趋势。

这些扰动用于展示分区异构控制逻辑，而不是直接模拟真实驱动器或真实涡结构。

## 8. 空化风险判断

当局部相对压力低于用户设置的阈值时，平台将该区域标记为潜在空化风险区。该判断是阈值可视化，不包含蒸汽压模型、多相流模型或气泡动力学。

## 9. 与真实 CFD 的区别

本平台：

- 二维；
- 轻量；
- 经验速度场；
- 伯努利压力估算；
- 强调交互、趋势和专利表达。

真实 CFD 通常需要：

- 三维几何和网格；
- 边界条件；
- 湍流模型；
- 空化或多相流模型；
- 收敛和网格无关性验证；
- 更高计算成本。

因此，本平台适合演示“分区主动控制为什么可能改善流场趋势”，不适合直接输出工程定量结论。
